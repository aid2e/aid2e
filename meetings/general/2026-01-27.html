AID2E bi-weekly general meeting

Meeting: January 27, 2026

Attendees: Meifeng Lin, Cynthia Nunez, Derek Anderson, Cristiano Fanelli, Hemalata Nayak, Karthik Suresh
  • Announcements
    • The group discussed extending Thursday technical meetings to cover both Phase I and Phase II work.
    • The team also planned to restructure notes and documentation on the AID2E website.
    • CHEP status and near-term note restructuring were both flagged as follow-up items.
  • Phase I
    • Migration work for the DTLZ2 example and workflow management using JobLibRunner was reported as complete.
    • The next step was to extend the same workflow model to SlurmRunner and PanDARunner.
    • Phase I documentation was expected to evolve alongside that migration.
  • Phase II
    • The group reviewed three major Phase II task areas.
    • Task Area 1 covered agentic workflows for code generation, pipeline assistance, and monitoring, including retries and PanDA integration.
    • The first action item was a state-of-the-art literature review on frameworks such as LangGraph, Colmena, Microsoft AutoGen, and ADK.
    • Task Area 2 covered bidirectional translation between software stack representations, with emphasis on speed, reversibility, and tolerance handling.
    • Task Area 3 focused on calibration and optimization tasks beyond detector design, including CLAS12 RICH alignment and future calorimeter calibration workflows.
    • The group agreed that Phase II should build on a stable AID2E 1.0 base rather than branching too early into disconnected prototypes.
  • AOB
    • A Phase I reference write-up remained an open request.
    • The paper to-do list still needed figures, task assignments, and a clearer Overleaf skeleton.
